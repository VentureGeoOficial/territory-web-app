# LGPD - Transparência e Tratamento de Dados

## Dados coletados
- Dados de conta: nome, e-mail, identificadores de login.
- Dados de uso: territórios, ranking, interações sociais.
- Dados opcionais: foto de perfil, plano de fundo e preferências de notificação.

## Finalidade de uso
- Autenticação e segurança da conta.
- Prestação do serviço (mapa, ranking, social).
- Comunicação de alertas no app, e-mail ou WhatsApp conforme consentimento.

## Proteção e governança
- Senhas tratadas por hash no provedor de identidade (Firebase Auth).
- Regras de acesso no Firestore para reduzir exposição de PII.
- Registro de aceite de Termos e Política com versão.

## Direitos do titular
- Atualização de dados pessoais na área de conta.
- Exclusão de conta com confirmação segura e reautenticação.
- Canais de suporte para dúvidas e solicitações.
