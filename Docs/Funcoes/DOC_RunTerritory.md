# DOC_RunTerritory

**Ficheiro:** [`lib/territory/run-territory.ts`](../../lib/territory/run-territory.ts)

- `validateRunTrack` — regras mínimas de pontos, duração, distância, fraccionamento dentro de Suzano.
- `createTerritoryFromRunTrack` — gera `Territory` a partir de pontos GPS e territórios existentes.
- `RUN_LINE_SIMPLIFY_TOLERANCE` — simplificação Douglas-Peucker.

Usado no mapa e no servidor (validação pré-transação na API).
